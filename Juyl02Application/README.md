# Juyl02Application 학습 정리

- 작성일: 2026-07-02
- 프로젝트: `Juyl02Application`

## 개요
이 저장소는 안드로이드 UI 레이아웃과 기본 이벤트 처리, 배치 방식, 커스텀 버튼 스타일 등을 연습하기 위한 여러 개의 실습 모듈로 구성되어 있습니다. 각 모듈은 독립적인 앱 형태로 만들어져 있으며, 하나의 주제를 집중적으로 학습할 수 있도록 구성되어 있습니다.

## 모듈별 학습 내용 요약

### `app`
- 기본 안드로이드 앱 구조를 확인할 수 있는 모듈입니다.
- `LinearLayout`과 `layout_weight`를 활용한 기본 화면 구성을 학습합니다.
- `enableEdgeToEdge()`와 `WindowInsets`를 이용한 최신 화면 처리 방식도 함께 확인할 수 있습니다.

### `dogcatshow`
- `ViewBinding`을 사용하여 뷰를 안전하게 참조하는 방법을 학습하는 모듈입니다.
- 버튼 클릭에 따라 이미지가 바뀌도록 구성되어 있어, 간단한 이벤트 처리와 `ImageView`의 가시성 제어를 익힐 수 있습니다.
- `FrameLayout`으로 이미지를 겹쳐 배치하는 방식도 확인할 수 있습니다.

### `gravityexam`
- `RelativeLayout`을 활용한 상대 위치 배치를 학습하는 모듈입니다.
- 중앙 기준 뷰를 중심으로 주변 요소를 정렬하는 규칙을 연습할 수 있습니다.
- `above`, `below`, `toStartOf`, `toEndOf`, `alignParent*` 같은 속성들의 사용법을 익히는 데 적합합니다.

### `phonepad`
- 전화기 숫자패드 UI를 구현한 모듈입니다.
- `TableLayout`과 `TableRow`로 숫자 버튼을 격자 형태로 배치하는 방법을 학습합니다.
- 둥근 버튼 스타일을 커스텀 drawable(`round_button.xml`)로 적용하여 숫자패드형 UI를 구성했습니다.
- 하단에는 영상통화, 다이얼링, 숫자삭제 버튼을 추가해 실제 전화 앱에 가까운 형태로 만들었습니다.

### `realativelayout`
- `RelativeLayout`의 다양한 정렬 규칙을 다시 연습하는 모듈입니다.
- 기준 뷰를 중심으로 여러 사각형을 상하좌우에 배치하는 구조를 통해 상대 배치 개념을 익힐 수 있습니다.
- 부모 기준 정렬과 형제 뷰 기준 정렬을 함께 이해하는 데 도움이 됩니다.

## 정리
이 프로젝트는 안드로이드 UI 레이아웃 학습을 중심으로,
- 기본 레이아웃 구성
- 이미지/버튼 이벤트 처리
- TableLayout 기반 키패드 UI
- RelativeLayout 상대 배치
- 커스텀 버튼 스타일 적용

같은 실습 내용을 단계적으로 익힐 수 있도록 구성된 학습용 프로젝트입니다.
