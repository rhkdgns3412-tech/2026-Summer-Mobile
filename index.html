Samsung Music 벤치마킹 프로젝트

프로젝트 소개

삼성 뮤직(Samsung Music)을 벤치마킹하여 Android에서 로컬 음악을 관리하고 재생할 수 있는 음악 플레이어 애플리케이션을 개발한다.

프로젝트 선정 이유

  • Android의 다양한 기능(MediaPlayer, RecyclerView, Room 등)을 활용할 수 있다.
  • 실생활에서 많이 사용하는 애플리케이션이라 사용성이 높다.
  • CRUD, 파일 관리, UI 구현 등 모바일 프로그래밍의 핵심 기술을 학습할 수 있다.
  • 기능 확장이 쉬워 프로젝트 완성도를 높일 수 있다.

구현 예정 기능

음악 재생

  • 음악 재생 / 일시정지
  • 이전 곡 / 다음 곡
  • 현재 재생 목록
  • 좋아요 표시
  • 재생목록 생성 및 관리
  • 곡 정보 확인
  • 볼륨 및 음향 조절
  • 셔플 및 반복 재생

음악 관리

  • 좋아요한 음악 목록
  • 사용자 플레이리스트 관리
  • 앨범별 분류
  • 아티스트별 분류
  • 폴더별 분류

개발 환경

  • Android Studio
  • Kotlin
  • Room Database
  • MediaPlayer / ExoPlayer
  • RecyclerView
