# 삼성 뮤직 벤치마킹 프로젝트

## 프로젝트 개요
기존 삼성 뮤직(Samsung Music) 애플리케이션을 벤치마킹하여 로컬 음악을 관리하고 재생할 수 있는 Android 음악 플레이어를 구현한다.



---

# 주요 기능

## 1. 음악 재생

### 1.1 음악 재생 화면
- 현재 재생 중인 곡의 제목, 아티스트, 앨범아트 표시
- 재생 시간 및 전체 재생 시간 표시
- SeekBar를 이용한 재생 위치 이동

### 1.2 재생 제어
- 음악 재생
- 일시정지
- 이전 곡
- 다음 곡
- SeekBar를 통한 원하는 위치로 이동

### 1.3 현재 재생 목록
- 현재 재생 대기 중인 음악 목록 확인
- 목록에서 원하는 곡 선택 후 즉시 재생

### 1.4 좋아요 기능
- 현재 재생 중인 곡 즐겨찾기(좋아요) 등록
- 좋아요 해제

### 1.5 재생목록 생성 및 관리
- 새로운 재생목록 생성
- 현재 곡을 원하는 재생목록에 추가
- 재생목록에서 음악 제거

### 1.6 곡 정보 보기
- 제목
- 아티스트
- 앨범명
- 재생 시간
- 파일 경로
- 파일 크기

### 1.7 음향 조절
- 시스템 볼륨 조절
- 음향 효과 설정

### 1.8 재생 모드
- 일반 재생
- 랜덤(셔플) 재생
- 한 곡 반복
- 전체 반복

---

# 2. 플레이리스트 관리

## 2.1 좋아요 목록
- 좋아요 표시한 음악만 모아서 조회
- 좋아요 해제

## 2.2 사용자 플레이리스트
- 플레이리스트 생성
- 플레이리스트 이름 수정
- 플레이리스트 삭제
- 음악 추가
- 음악 삭제
- 플레이리스트 전체 재생

---

# 3. 음악 분류

## 3.1 앨범별 분류
- 앨범 목록 조회
- 앨범 선택 시 해당 앨범의 음악 목록 표시

## 3.2 아티스트별 분류
- 아티스트 목록 조회
- 아티스트 선택 시 해당 아티스트의 음악 목록 표시

## 3.3 폴더별 분류
- 저장된 폴더 목록 조회
- 폴더 선택 시 해당 폴더의 음악 목록 표시

---

# 추가 기능 (선택 구현)

- 음악 검색
- 최근 재생한 음악
- 최근 추가한 음악
- 재생 중인 음악 알림(Notification)
- 백그라운드 재생
- 잠금화면 음악 제어
- 다크 모드 지원
